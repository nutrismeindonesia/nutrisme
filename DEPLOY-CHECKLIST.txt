B. VERIFIKASI GOOGLE APPS SCRIPT
--------------------------------
Meng-upload apps-script.gs ke GitHub TIDAK memperbarui Google Apps Script.
Salin seluruh isi apps-script.gs ke editor Google Apps Script yang dipakai.

Di editor Apps Script, cari teks:

  function doGet()
  function doPost(e)

Jalankan setupNutrisme satu kali dan setujui izin.

Kemudian:
Deploy > Manage deployments > Edit > Version: New version
Execute as: Me
Who has access: Anyone

C. HEALTH CHECK WAJIB
---------------------
Buka URL Web App yang berakhir /exec di mode incognito.
Hasil yang benar harus memuat:
  "status":"ok"
  "version":"2026-07-17-2"

Jika masih muncul:
  Script function not found: doGet
maka Web App masih memakai deployment lama atau URL dari project yang berbeda.
Jangan lanjut test form sebelum health check benar.
